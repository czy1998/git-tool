# git-tool

本工具是用于帮助开发者更好的管理分支。

可以查看所有的本地分支以及它们最新的 5 条（默认配置是 5 条）commit 记录，然后可以选择部分分支进行操作（删除远程分支/删除本地分支）

## 如何使用

```sh
# 全局安装工具
npm i -g git-branch-tool

# 进入需要操作的项目目录，开始操作
gtool
```

> ⚠️ 由于工具中使用了部分 ES6 语法，建议 node 版本 > 20

## 技术栈

- typescript
- ora 更优雅的处理命令行提示、添加动画
- biome 样式检查、自动修复工具
- yargs-parser 接收解析命令行参数
- @inquirer/prompts 一组常用的交互式命令行用户界面

## 功能清单

- [x] 批量删除本地分支、远程分支
- [x] 对话形式交互
